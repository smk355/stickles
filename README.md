# Stickles

Stickles is a modern catalogue website showcasing **bracelets, phone charms, and stickers**.  
It is designed to present products in a clean, visually appealing, and easy-to-browse format.

## Project Overview

Stickles serves as a digital catalogue for accessories, allowing users to explore different product categories with a smooth and responsive UI.
